# Snippetbox

A tutorial project written in Go, completed by following the Let's Go book.

## Table of Contents:

- [Project Structure](./docs/project-structure.md)

## Development Mode

**Requirements:**

- Go v1.20
- [air](https://github.com/cosmtrek/air) for hot reloading

**Steps:**

```
go mod tidy
cd cmd/web
air
```
